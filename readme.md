# TimeTogether

## 📌 프로젝트 소개
TimeTogether는 팀과 조직의 일정을 효율적으로 관리할 수 있는 협업 일정 관리 웹 서비스입니다.

## 🛠 기술 스택

### Frontend
- React 19.0.0
- TypeScript
- Tailwind CSS v4.0

### Backend
- Spring Boot 3.4.3
- Gradle

## 💻 개발 환경

### 백엔드 실행
```bash
cd BE
./gradlew clean build bootRun
```

### 프론트엔드 실행
```bash
cd FE
npm install
npm run dev
```
